# Information Processing Systems 2025-2026

Questo progetto contiene il file `main.py` relativo al corso di Sistemi di Elaborazione delle Informazioni per l'anno accademico 2025-2026.

## Struttura del progetto

- `main.py`: File principale del progetto. Contiene il codice sorgente principale.

## Come eseguire il codice

1. Assicurati di avere Python installato (versione 3.8 o superiore consigliata).
2. Apri un terminale nella cartella del progetto.
3. Esegui il comando:
   ```bash
   python main.py
   ```
